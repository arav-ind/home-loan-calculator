# Home Loan Amortization Calculator

React application to calculate Home Loan EMIs and view a detailed amortization schedule. This app supports multiple partial prepayments (One-time, Monthly, Yearly) to help users understand how prepayments can reduce their loan tenure and interest.

## Live Demo
**[Launch App](https://arav-ind.github.io/home-loan-calculator/)**

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/arav-ind/home-loan-calculator.git
    ```
2.  Install dependencies:
    ```bash
    pnpm install
    ```
3.  Run development server:
    ```bash
    pnpm run dev
    ```
